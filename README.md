# daremall

thymeleaf & springboot를 활용한 shopping mall app

## stack
- thymeleaf
- springboot
- h2

## using api
- SMS 서비스 : coolsms (https://coolsms.co.kr)
- 도로명 주소 : daum postcode (https://postcode.map.daum.net/guide)
- 카카오페이 결제 : iamport (https://www.iamport.kr)

## 기능

사용자 기능과 관리자 기능으로 구분

### 사용자 기능

- 주문 및 결제
- 관심 상품 등록 및 장바구니
- 회원 정보 관리

### 관리자 기능

- 주문 관리
- 상품 관리
- 광고 관리
- 회원 관리
- 통계 확인
